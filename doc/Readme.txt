===============================================================================

　WChat - チャットツール

===============================================================================


■ソフトの概要

　クライアント・サーバー方式のチャットツールです。
　ファイル共有・転送機能として、各クライアントのローカルディスクをブラウザで閲覧する機能が付いています。
　セキュリティは特に考慮していません。ＬＡＮ内での利用をおすすめします。

　ソースコードを公開しています。


■動作環境

　Windows 10
　確認していませんが Windows 7/8/8.1 でも動くかもしれません。


■フォルダ構成

　WChat_Client　...　クライアント
　WChat_Server　...　サーバー


■インストール方法

　アーカイブ内の WChat_Client, WChat_Server をローカルディスク上の任意の場所にコピーして下さい。


■アンインストール方法

　レジストリなどは一切使っていません。
　ファイルを削除するだけでアンインストールできます。


■使い方

　各フォルダの Readme.txt を参照して下さい。
